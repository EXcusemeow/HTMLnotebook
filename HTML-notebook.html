类class 、id

同类名可以由多个 HTML 元素使用，而 id 名称只能由一个元素使用
#id -- .class召唤 /id传送(href=#）
js使用getElementById
HTML的ID、class和name属性可以使用中文，但并不推荐这样做。

id 属性用于给 HTML 元素定义一个唯一的标识符，id 在 JavaScript 中也用于获取该元素的引用，通过 document.getElementById() 方法可以获取一个元素的引用。
name 属性则用于定义 HTML 表单元素的名称。在表单提交时，名称将被用于生成 HTTP 请求，后端可以通过该名称获取表单数据。
class 属性则用于定义元素的类，一个元素可以同时有多个类。类可以用来为元素定义样式，也可以用于 JavaScript 中通过 document.getElementsByClassName() 方法获取所有具有该类的元素。

表单form
form action="xxx.asp或.php" method=

input form="form的id" 输入字段位于 HTML 表单之外（但仍属表单）：
autocomplete 属性表单会根据用户之前输入的值自动填写值。
action 属性指定表单处理程序
input formaction 覆盖form的action
如果省略 action 属性，则 action 会被设置为当前页面。
method 属性指定提交数据的 HTTP 方法。 取值有 GET 和 POST。
GET ，表单数据会附加在 URL 的末尾，形成 http://example.com/?name1=value1&name2=value2 的形式。
GET 少量数据查询和数据展示
POST ，表单数据会作为请求体发送，不会附加在 URL 上。
POST 重要数据提交和修改
